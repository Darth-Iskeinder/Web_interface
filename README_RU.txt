Директория содержит веб-интерфейс управления базой данных.

Для запуска программы выполните следующие шаги:
1) Перенесите дамп базы данных на свой сервер (dump_database.rar)
2) Перенесите файлы в рабочую директорию своего сервера
3) Запустите сервер (localhost)  
4) Пройдите процедуру авторизации (login = "admin", password = "123")

После авторизации вы получите доступ к списку зарегистрованных пользователей разделенных 
постраничной навигацией, а так же возможностью добавления и редактирования пользователей, 
просмотра данных по каждому пользователю отдельно.
