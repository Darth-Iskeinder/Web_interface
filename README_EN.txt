The directory contains the database management web interface.

To start the program, follow these steps:
1) Transfer the database dump to your server (dump_database.rar)
2) Transfer the files to the working directory of your server
3) Start the server (localhost)
4) Go through the authorization procedure (login = "admin", password = "123")

After authorization, you will get access to the list of registered users divided
page navigation, as well as the ability to add and edit users,
viewing data for each user separately.
